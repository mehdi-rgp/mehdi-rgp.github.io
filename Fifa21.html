Fifa21 Data Cleaning and Transformation

Emphasizing the importance of data quality
Performed using Excel Power Query

Project Overview

The project highlights the vital role of data cleaning, adhering to the 'garbage in, garbage out' principle. I worked with a dataset from EA Sports' FIFA21, sourced from Kaggle, which was very messy and full of formatting issues, rendering it unsuitable for analysis in its original state. Containing over 19,000 rows, this project demonstrates the essential process of transforming raw, disorganized data into a clean and refined format, crucial for any meaningful analysis.

Tools Used

I employed Excel Power Query for this project, a robust tool for data manipulation. Power Query stands out in handling large and complex datasets more efficiently than traditional Excel functions, making it an ideal choice for this extensive data cleaning task.

Project Files

The outcomes of the data cleaning process can be accessed online at this link. To further explore or modify the queries, download the .xlsx and .csv files. Note that these files require the Excel desktop version for full functionality, as Office Online does not support Power Query.

Data Cleaning Highlights

  • Numerical Conversions: Columns 'Age', 'OVA', and 'POT' were converted to numerical formats.
  • Date Formatting: Updated 'Joined' and 'Loan Date End' to a uniform date format.
  • Column Name Standardization: Renamed columns for clarity, removing special characters.
  • Contract Column Transformation: Reorganized the 'Contract' column into 'Contract Type', 'Contract Start', and 'Contract End'.
  • Numerical Data Formatting: Standardized units in height and weight columns, and converted monetary values to a numerical format.
  • Character Removal and Conversion: Removed extraneous characters from specific columns and converted them to numbers.
  • Whitespace Elimination: Cleared extra spaces in the 'Club' column.
  • Removing Unnessary Columns: Columns such as 'LongName', 'photoUrl, 'playerUrl' that are not required in future analysis will be removed to have a more consice data.
These steps transformed the FIFA21 dataset into a more structured and analyzable format, demonstrating the critical role of data cleaning in data analytics.
